## How to register

### Remote trigger (claude.ai/code/scheduled)
1. Go to Code tab -> Scheduled -> New Task -> New local task (Desktop) or open [claude.ai/code/scheduled](https://claude.ai/code/scheduled)
2. Name: <Name of Task in the trigger files>
3. Schedule: Pick frequency and time <Schedule in the trigger files>
4. Repo: local project folder, tick "worktree"
5. Paste the prompt from the trigger file
6. Choose "Auto-accepct edits" in the "Ask permissions" selector
7. Choose your model (optional)
8. Click "Create Task"
